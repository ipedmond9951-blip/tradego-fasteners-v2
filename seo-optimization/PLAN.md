# TradeGo SEO优化计划 (40轮迭代)

## 第一阶段：SEO规则学习 + 竞品研究 (1-5轮)

### 1.1 SEO核心规则学习
**目标关键词类型:**
- 产品词: `drywall screws`, `self drilling screws`, `hex bolts`, `IBR nails`
- 商业词: `manufacturer`, `wholesale`, `supplier`, `factory`, ` exporter`
- 长尾词: `drywall screws for wood framing`, `self drilling screws stainless steel`
- 地区词: `from China`, `Zimbabwe`, `South Africa`, `Dubai`

**SEO评分维度:**
1. On-Page SEO (40分)
   - Title标签优化 (10分)
   - Meta描述 (10分)
   - H标签结构 (10分)
   - 图片Alt文本 (10分)

2. Technical SEO (40分)
   - 页面速度 (10分)
   - 移动端适配 (10分)
   - 结构化数据 (10分)
   - 内链策略 (10分)

3. Off-Page SEO (20分)
   - 外链质量 (10分)
   - 社媒信号 (5分)
   - 品牌提及 (5分)

### 1.2 竞品研究列表

**主要竞品网站:**
1. Alibaba.com ( fastener suppliers)
2. Made-in-China.com (中国制造网)
3. Globalsources.com
4. 同行排名高的制造商网站

**研究内容:**
- Title格式: 如何写产品标题
- Meta描述: 如何包含关键词
- 图片命名: 如何优化文件名
- 产品描述: 如何写高质量内容
- 关键词布局: 如何分布关键词
- 结构化数据: 如何使用Schema

### 1.3 产品上传学习来源

**Alibaba学习要点:**
- 产品标题格式: `Main Keyword + Model/Spec + Material + Surface Treatment + Quantity`
- 产品图片: 主图白底、产品细节、应用场景
- 产品描述: 详细规格表+FAQ+优势说明
- 关键词标签: 3-5个相关关键词

**Made-in-China学习要点:**
- 产品分类结构
- 详情页SEO布局
- 站内优化技巧

## 第二阶段：竞品分析执行 (6-10轮)

### 2.1 抓取分析任务

```bash
# 竞品URL列表
1. https://www.alibaba.com/product-detail/xxx.html
2. https://www.made-in-china.com/product/xxx.html
3. 谷歌搜索 "drywall screws manufacturer" 前5名网站
```

### 2.2 分析维度

| 分析项 | 记录内容 |
|--------|----------|
| Title | 完整标题文本 |
| Title长度 | 字符数 |
| Title关键词 | 包含的关键词 |
| Meta描述 | 前160字符 |
| H1标签 | 页面主标题 |
| 图片Alt | alt文本样本 |
| 关键词密度 | 内容中关键词出现频率 |
| 内链数量 | 页面内链数量 |
| 外链数量 | 页面外链数量 |
| 页面长度 | 内容字数估计 |
| 结构化数据 | 是否使用JSON-LD |

## 第三阶段：网站SEO优化 (11-20轮)

### 3.1 技术SEO修复

| 问题 | 修复方案 | 优先级 |
|------|----------|--------|
| 图片Alt缺失 | 批量生成alt文本 | P0 |
| Meta描述缺失 | 为每个页面生成描述 | P0 |
| 结构化数据缺失 | 添加Product/Organization Schema | P1 |
| 内链不足 | 添加相关产品链接 | P1 |
| 页面速度 | 图片压缩+懒加载 | P2 |

### 3.2 内容SEO优化

| 页面类型 | 优化项 | 目标 |
|---------|--------|------|
| 首页 | Title/Meta/H标签 | 提升品牌词排名 |
| 产品页 | 描述/Alt/内链 | 提升产品词排名 |
| 博客文章 | 内容质量/关键词 | 提升长尾词排名 |
| 分类页 | Title/Meta | 提升分类词排名 |

## 第四阶段：产品上传优化 (21-30轮)

### 4.1 产品标题格式

```
{主关键词} {规格} {材质} {表面处理} {品牌} {型号}

示例:
Drywall Screws 3.5x25mm Carbon Steel Zinc Plated for Wood Framing
```

### 4.2 产品描述模板

```html
<h1>{产品名称}</h1>

<h2>Product Specifications</h2>
<table>...规格表...</table>

<h2>Product Features</h2>
<ul>...产品特点...</ul>

<h2>Applications</h2>
<p>...应用场景...</p>

<h2>Packaging & Shipping</h2>
<p>...包装发货...</p>

<h2>FAQ</h2>
<p>...常见问题...</p>
```

### 4.3 图片优化标准

| 图片类型 | 尺寸要求 | 命名格式 | Alt文本 |
|---------|----------|----------|---------|
| 主图 | 800x800+ | `product-name-main.jpg` | `{产品名} - 正面图` |
| 细节图 | 800x600+ | `product-name-detail-1.jpg` | `{产品名} - 螺纹细节` |
| 应用图 | 800x600+ | `product-name-application.jpg` | `{产品名} - 应用场景` |

## 第五阶段：外链建设 (31-40轮)

### 5.1 外链来源

1. 目录提交 (DMOZ, Yelloh)
2. 行业目录 (中国制造网目录)
3. 社媒档案 (LinkedIn, Facebook, Twitter)
4. 论坛签名 (行业论坛)
5. 客座文章 (行业博客)

### 5.2 外链质量评估

| 质量等级 | DA/DR | 操作 |
|---------|-------|------|
| 高 | 50+ | 客座文章、采访 |
| 中 | 20-50 | 目录提交、社媒档案 |
| 低 | <20 | 论坛、评论 |
