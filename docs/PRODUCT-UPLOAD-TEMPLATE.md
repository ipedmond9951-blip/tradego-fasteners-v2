## Product Title Template

**Format:**
```
[Product Name] + [Key Specification] + [Application] + [Certification] + [Brand/Origin]
```

**Examples:**
```
Drywall Screws 3.5x25mm Bugle Head Fine Thread for Gypsum Board Construction ISO 9001
Self-Drilling TEK Screws #8x1/2 inch Hex Washer Head Zinc Plated Metal Roofing Fastener
Hex Bolt M10x50 Grade 8.8 High Tensile Steel Zinc Plated DIN931 ISO 9001 China Manufacturer
IBR Roofing Nails 3.05x75mm Umbrella Head Smooth Shank Hot Dip Galvanized SABS Certified
```

**Title Rules:**
- Length: 50-80 characters
- Include: Product name, size/spec, material, surface finish, standard, application
- Include: Certification (ISO 9001, CE, SABS)
- Include: "Manufacturer" or "Factory" or "Wholesale"
- Primary keyword at the beginning
- Use numerical values (3.5mm not "three point five millimeters")

## Product Images (6-8 images)

### Image Requirements:
1. **Main Image**: Product on white background, 800x800px minimum
2. **Side View**: Show profile/shape clearly
3. **Close-up/Detail**: Thread detail, head style, point type
4. **Size Reference**: With ruler or coin for scale
5. **Packaging**: How product is packaged (bulk bag, box)
6. **Application**: Product in use (construction site, roofing)
7. **Factory**: Production facility (if available)
8. **Certification**: ISO/CE/SABS certificates

### Image Naming:
```
drywall-screws-3.5x25mm-bugle-head-main.jpg
drywall-screws-3.5x25mm-bugle-head-side-view.jpg
drywall-screws-3.5x25mm-bugle-head-closeup.jpg
drywall-screws-3.5x25mm-size-reference.jpg
drywall-screws-3.5x25mm-packaging.jpg
drywall-screws-3.5x25mm-application-construction.jpg
```

### Image Alt Text Format:
```
[Product Name] - [Specification] - [Application] - TradeGo Fasteners Manufacturer
Example: "Drywall Screws - 3.5x25mm Bugle Head Fine Thread - Gypsum Board Installation - TradeGo Fasteners Manufacturer"
```

## GEO Optimization for Product Pages

### Key Principles:
1. **Specific numbers**: Use exact measurements (3.5mm not "~3.5mm")
2. **Application context**: Describe where/how product is used
3. **Trust signals**: Mention certifications, factory experience
4. **FAQ structure**: Include common questions with specific answers
5. **Structured data**: Include Product Schema (JSON-LD)

### For AI Discoverability:
- Include "problem-solution" format
- Use "How to" language
- Add comparison tables
- Include common mistakes to avoid
- Add "expert tip" sections
